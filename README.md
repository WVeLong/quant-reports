# quant-reports

A curated collection of quantitative research reports, covering factor mining, high-frequency factor construction, portfolio construction, risk models, and symbolic/GP factor mining.

本仓库包含“量化研报阅读计划”静态网页库。

入口：[`index.html`](./index.html)

目录：

- `categories/`: 按阅读主题分类的网页入口
- `notes/`: 每篇报告的阅读问题页
- `pdf/`: 按分类整理的完整研报 PDF
- `download_manifest.csv`: 匹配和下载清单
- `manifest.json`: 网页索引数据
